# Альтернативная задача

Дана последовательность из 8 цифр. Нужно расставить между ними знаки арифметических операций так, чтобы получилось число 100.

Возможные операции: + - * /

Пример: 98765432

Результат:

98+7+6-5-4*3/2=100

98+7-6-5+4*3/2=100

98+7-6*5/4/3*2=100

98+7*6+5-43-2=100

98+7*6-5/4*32=100
...
